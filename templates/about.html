Project Mission

Project goal is to detect and visualize fraudulent transactions using Machine Learning and a real-time web dashboard. This project empowers businesses to act quickly and make data-driven decisions with confidence.

Tech Stack

Python
Flask
HTML5
CSS5
Bootstrap
Matplotlib
Javascript
Pandas & Sklearn

How It Works

  1. Step 1: The user uploads transaction data or submits via form.
  2. Step 2: The backend Flask app runs the trained ML model to detect fraud probability.
  3. Step 3: A prediction result is shown with fraud probability.
  4. Step 4: The dashboard displays key metrics like fraud % and country trends with charts.
  5. Step 5: User can explore visual reports generated with Chart.js and Matplotlib.
